# scrcpy Setup

#### - Install Scrcpy Latest Version

```
sudo apt remove scrcpy

sudo apt install git ffmpeg libsdl2-2.0-0 adb

git clone https://github.com/Genymobile/scrcpy

git clone https://github.com/Genymobile/scrcpy

cd scrcpy

sudo apt install meson ninja-build
meson setup build
cd build
ninja

sudo ninja install

scrcpy --version

```
